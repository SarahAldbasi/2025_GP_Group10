## Launching Instructions for Hakkim SmartWatch App

To run the Hakkim SmartWatch app using Android Studio:

1. Download the `HakkimSmartWatch.zip` file from this repository.
2. Extract the ZIP file to a folder on your local machine.
3. Open **Android Studio**.
4. Click on **"Open an existing project"** and select the extracted project folder.
5. Let Android Studio finish syncing and building the project.
6. Set up a **Wear OS emulator**:
   - Go to **Device Manager** > **Create Virtual Device**.
   - Choose **Wear OS** and select a device ("Wear OS Small Round").
   - Download the recommended system image and create the emulator.
7. Run the project:
   - Make sure the emulator is running.
